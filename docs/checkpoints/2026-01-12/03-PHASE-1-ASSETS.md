# Phase 1 Assets – Website Visuals

## Required Assets
### Homepage
- Hero background (city / executive / security context)
- Service section imagery (3–4 cards)
- Trust section icons (licensed, insured, vetted)
- CTA background variants

### Style
- Natural lighting
- Real environments
- No over-processed HDR
- Avoid obvious AI artifacts

## Output Formats
- PNG (primary)
- WEBP (optimized)
- Standard widths:
  - 2560px (hero)
  - 1440px
  - 1024px
  - 768px
